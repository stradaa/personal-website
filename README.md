# Alejandro Estrada Berlanga Website

Static personal portfolio designed for GitHub Pages. It combines neuroengineering research material with a small section for sharing beautiful nature images.

## Edit The Site

- Update `index.html` with your name, bio, links, publications, projects, and captions.
- Replace `assets/hero-nature.png` with your own image when ready.
- Add shared nature images to `assets/photos/`.
- Add full-resolution downloadable files to `assets/downloads/` if you want separate download versions.
- Keep large image files compressed enough that the repository stays comfortably under GitHub Pages limits.

## Publish With GitHub Pages

1. Push this folder to a GitHub repository.
2. In the repository, open **Settings > Pages**.
3. Set the source to deploy from the main branch root.
4. Save, then use the generated GitHub Pages URL or connect a custom domain.

No build step is required.
